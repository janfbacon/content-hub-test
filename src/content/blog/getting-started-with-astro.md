# Getting Started with Astro

Astro is a modern web framework that delivers lightning-fast performance by default. Its unique **Islands Architecture** allows you to mix static content with interactive components, giving you the best of both worlds.

## What Makes Astro Special?

1. **Zero JS by Default**: Astro ships zero JavaScript to the client unless you explicitly opt-in
2. **Islands Architecture**: Only hydrate the components that need interactivity
3. **Framework Agnostic**: Use React, Vue, Svelte, or any other framework you prefer
4. **Content Collections**: Built-in support for managing Markdown and MDX files

## Getting Started

To create a new Astro project, simply run:

```bash
npm create astro@latest
```

This will guide you through setting up your project with all the features you need.
